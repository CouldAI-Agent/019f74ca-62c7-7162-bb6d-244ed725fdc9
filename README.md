# Fresher Resume Builder

A simple, responsive Flutter application that helps recent graduates create a clean and professional resume. Users can enter their personal details, professional summary, education, skills, and projects, and generate a polished resume preview.

## Features
- Clean and intuitive form for entering resume data.
- Sections for Personal Details, Education, Skills, and Projects.
- Live, formatted resume preview screen.
- Responsive design that works well on mobile, web, and desktop platforms.

## Tech Stack
- Flutter
- Dart
- Material 3 Design

## Setup / Run
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to fetch dependencies.
3. Start the app on your preferred device with `flutter run`.
